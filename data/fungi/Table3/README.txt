table_after_filtering.csv > This file (to be opened as a spreadsheet in, e.g. Excel) presents the number of sequences in each ASV, for each sample, and the taxonomy of that ASV. 

table_after_filtering.biom > This file presents the number of sequences in each ASV, for each sample, and the taxonomy of that ASV in a biom format. This format allows for storage of different amounts of samples, it has a smaller memory footprint and it is designed to be a general-use format.
